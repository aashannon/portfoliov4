Other Projects

Part portfolio, part time capsule; this is where code met creativity. These aren’t just side projects, they’re stepping stones. From early experiments to polished creations, each one shaped how I think, design, and build.

Technical Projects

Here’s where things get nerdy (in the best way). I built my own search engine from scratch and coded a Connect 4 game with a rough-around-the-edges AI. These projects show what happens when curiosity meets code.

Search Engine

A fully functional search engine built from scratch using Python. It crawls web pages, indexes content, and ranks results using basic relevance algorithms. This project challenged me to think critically about data structures, information retrieval, and backend logic; all while building a usable tool from the ground up.

Tags: Python, Web Crawling, Indexing, Search Algorithms, Information Retrieval, Data Structures, Backend Development

Connect 4 with AI Player

A digital version of Connect 4 built with Python and a basic AI opponent. The AI uses a simple heuristic to block and win moves, introducing strategic gameplay without complex algorithms. This project sharpened my skills in game logic, user interaction, and implementing decision-making with code.

Tags: Python, Game Development, Heuristic AI, Turn-Based Logic, User Interaction, Problem Solving

Not everything I make sits still. These animations and multimedia pieces are where I let design move, breathe, and play. Whether it’s motion graphics or interactive visuals, each project explores a different layer of how users experience motion.
